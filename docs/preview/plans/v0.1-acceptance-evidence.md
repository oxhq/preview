# v0.1 Acceptance Evidence

Last verified in this repository on 2026-05-02.

## Commands

```bash
composer test
composer validate --strict
php -l config/preview.php
php -l src/**/*.php
php -l tests/**/*.php
```

## Current Proof

- Full suite: 63 tests, 249 assertions.
- Generic, HMAC, and Stripe capture-to-fixture-to-test flows are covered by Testbench acceptance tests.
- Exact replay preserves local raw headers and body while show/fixture metadata remains redacted.
- HMAC and Stripe resign replay refresh provider-valid signatures.
- Generated Pest-compatible tests load fixtures through relative paths where possible.
- Sensitive fixture payloads are written under `.local` and gitignored when the fixture root is inside a git repo.
- Live tunnel capture requires both `--live` and `preview.live_enabled=true`.

## Proof Boundary

This is package-internal Testbench proof, not hosted CI, package publication, or a fresh external Laravel app installation. Pest-compatible generated files are executed through the package acceptance shim because Pest is not a dev dependency of this package.
